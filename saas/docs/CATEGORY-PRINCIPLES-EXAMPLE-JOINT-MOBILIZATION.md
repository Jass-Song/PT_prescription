## 2. 핵심 원칙 (5개)

### 원칙 1 — 치료 전 평가가 먼저

- **아이콘**: 🎯
- **제목**: 치료 전 평가가 먼저
- **설명**: 시술 전 반드시 관절 가동 범위(ROM)와 통증 위치를 확인하세요. 치료 후 변화를 비교해야 효과를 판단할 수 있습니다.
- **왜 중요한가**: 기저선(baseline) 없이 시술하면 효과 유무를 판단할 수 없고, 분절 선택이나 Grade 조정의 근거가 사라진다. 매 세션 반복 평가가 Maitland Concept의 핵심 원리다.

### 원칙 2 — 저항 감지가 핵심 (R1 / R2)

- **아이콘**: 🖐️
- **제목**: R1/R2 저항 감지
- **설명**: R1은 처음 조직 저항이 느껴지는 지점, R2는 최대 저항 지점입니다. Grade I-II는 R1 이전, Grade III-IV는 R1-R2 사이에서 시행합니다.
- **왜 중요한가**: 저항 감지 없이 기계적으로 Grade를 적용하면 통증 우세 환자에게 과도한 자극을 주거나, 굳음 우세 환자에게 치료적으로 불충분한 자극을 주게 된다. 치료사의 손 감각이 Grade 결정의 1차 기준이다.

### 원칙 3 — Grade 선택 기준

- **아이콘**: 📊
- **제목**: Grade 선택 기준
- **설명**: Grade I-II: 통증이 주 증상일 때 (부드럽게). Grade III-IV: 관절 굳음이 주 증상일 때 (더 깊게). 처음에는 낮은 Grade로 시작해 반응을 봅니다.
- **왜 중요한가**: Grade 선택 오류는 가장 흔한 실수다. 굳음 환자에게 Grade II만 반복하면 효과가 없고, 급성 통증 환자에게 Grade IV를 바로 적용하면 증상이 악화된다. 통증 vs. 굳음의 주 증상 구별이 Grade 결정의 핵심 판단 기준이다.

### 원칙 4 — 시술 후 즉각 재평가

- **아이콘**: 🔄
- **제목**: 즉각 재평가
- **설명**: 3-5회 시행 후 ROM 변화와 통증 변화를 반드시 확인하세요. 변화가 없으면 방향, 분절, 또는 Grade를 조정합니다.
- **왜 중요한가**: Maitland Concept은 "치료-평가-재치료"의 순환 구조를 전제로 한다. 재평가 없이 동일 자극을 반복하는 것은 이 개념과 근본적으로 충돌한다. 재평가 결과가 다음 세션의 방향을 결정한다.

### 원칙 5 — 절대 금기 확인 (필수)

- **아이콘**: ⚠️
- **제목**: 절대 금기 확인
- **설명**: 골절, 종양, 심한 골다공증, 척수 압박 증상(양하지 저림·대소변 장애)이 있으면 시행하지 마세요.
- **왜 중요한가**: 환자 안전 — 특히 경추 가동술 시 척추동맥 혈액순환 장애(VBI) 스크리닝은 시술 전 필수다. 신경학적 결손(근력 저하, 반사 소실)이 확인된 경우 가동술 전 의사 협진이 원칙이다.

## 3. 이 접근법으로 치료하기 좋은 환자 프로파일

| 항목 | 내용 |
|------|------|
| 주 적응증 | 관절 가동 범위 제한을 동반한 경부통, 요통, 사지 관절 통증 |
| 이상적인 환자 상태 | subacute (2-12주) 또는 chronic (12주 이상) — 급성기에도 Grade I-II로 통증 조절 목적 적용 가능하나 주 적응증은 아급성 이후 |
| 특히 효과적인 증상 | 움직일 때 통증, 관절 뻣뻣함(hypomobility), 경추성 두통 |
| 이 접근법이 적합하지 않은 경우 | 과가동성(hypermobility), 급성 신경근병증(방사통 악화 위험), 골절·종양·불안정성 |

## 4. 다른 접근법과의 차이점

| 비교 접근법 | 차이점 | 언제 관절가동술을 선택하는가 |
|------------|--------|--------------------------|
| 멀리건 기법 (Mulligan) | 멀리건은 환자의 능동 움직임 중 치료사가 동시에 활주(glide)를 더하는 방식. 통증 없는 움직임이 즉각적으로 가능해야 적용 가능. | 환자가 능동 움직임 수행이 어려운 급성~아급성 단계, 또는 수동 평가에서 뚜렷한 저항이 확인된 경우 |
| 도수조작 (Manipulation / HVLAT) | 고속-저진폭(High Velocity Low Amplitude Thrust) 기법. 한 번의 충격으로 관절을 열어주는 방식. 가동술보다 즉각 효과가 크지만 절대적 적응증 선택이 더 까다로움. | 저항이 뚜렷하나 도수조작에 대한 환자 거부감이 있거나, 단계적 접근이 필요한 경우 관절가동술을 먼저 적용 |
| 신경가동술 (Neurodynamics) | 신경계 흥분성 저하를 목적으로 신경 슬라이딩·텐셔닝 기법을 사용. 관절 가동 범위보다 신경 증상(저림, 방사통) 완화에 특화. | 방사통·저림 없이 관절 ROM 제한과 국소 통증이 주 증상인 경우 관절가동술 선택 |

## 6. KMO 철학 정합성 체크

- [x] 통증 = 손상이 아님을 전제하는가? — Grade I-II는 신경계 조절 목적으로 작용하며, 통증 = 구조 손상 프레임을 전제하지 않음
- [x] 환자의 능동적 참여를 강조하는가? — 재평가 후 ROM 확인 및 가정운동 병행 권장
- [x] 공포 회피 유발 표현이 없는가? — 금기 설명 시 "시행하지 마세요" 수준으로만 기술, 공포 조장 없음
- [x] 생물심리사회적 관점과 충돌하지 않는가? — 통증 조절과 기능 회복을 동시에 목표로 하며 환자 반응 중심 접근
- [x] "Calm things down, and build things back up" 원칙과 일치하는가? — Grade I-II로 통증을 먼저 안정시키고, Grade III-IV로 기능 회복
- [x] KPM 프레임 (자세 불균형, 정렬 이상) 언급이 없는가? — 원칙 설명 전체에서 자세·정렬 프레임 없음
