# Red Flag 빠른 참조 카드 (PT 임상 안전망)

**작성일**: 2026-05-12
**대상**: 베타 참여 PT 선생님
**용도**: 환자 사전 스크리닝 1페이지 요약 — 인쇄 또는 휴대폰 즐겨찾기 권장
**출처**: APTA CPG (Neck 2017 / LBP 2021 / Hip 2025 / Knee 2018 / PFP 2019 / Shoulder 2013·2020·2025 / Ankle 2021) + KAOMPT 안전 가이드 + Kerry & Taylor *JOSPT* 2009 (CAD)
**철학**: KMO — "Calm things down, and build things back up." (Greg Lehman) · 두려움 자극 없이 안전한 의뢰 흐름.

---

## ✅ 사용법

1. 도수치료 시작 전 환자 인터뷰에서 아래 8 패턴을 확인합니다.
2. 해당 항목이 1개 이상이면 앱 화면의 **안전 확인 (선택)** 섹션에서 체크합니다.
3. 1개 이상 체크되면 추천이 일시 멈추고 의료 의뢰 안내가 표시됩니다.
4. 의뢰 결정은 PT 선생님의 임상 판단입니다. 본 카드는 보조 체크리스트입니다.

> **추천 시스템은 PT 임상 판단을 대체하지 않습니다.** 환자 상태가 애매하면 보수적으로 의뢰를 권합니다.

## 📋 8 패턴 × {증상 / 임상 의미 / 처치}

| # | 앱 UI 라벨 (한국어) | 임상 의미 (1줄) | 발견 시 처치 |
|---|---|---|---|
| 1 | **어지러움·구토·시야 이상** | VBI (척추동맥부전) 신호 — 복시·구음장애·기절 동반 시 특히 주의 | Cervical manipulation 보류 · 의료 의뢰 권고 |
| 2 | **안장 부위 감각 이상·대소변 조절 문제** | Cauda equina 신호 — 양측 하지 마비 동반 가능 | 도수치료 보류 · 즉시 의료 의뢰 권고 |
| 3 | **진행성 근력 약화·반사 변화** | Myelopathy 징후 (Hoffmann·Babinski 양성 등) | 도수치료 보류 · 의료 의뢰 권고 |
| 4 | **야간통 + 체중감소 + 발열** | 비기계적 통증 패턴 — 종양·감염 가능성 (50세 이상 신규 통증 포함) | 도수치료 보류 · 영상·혈액 검사 의뢰 권고 |
| 5 | **최근 외상 직후** | 골절·인대 파열 가능성 — 영상 진단 미실시 시 | 도수치료 보류 · 영상 진단 우선 권고 |
| 6 | **안정 시에도 강한 통증 + 진행성 악화** | 급성 reactive 상태 — 자극에 매우 민감 | Maitland Grade I 이하 또는 진정 우선 · 의료 의뢰 검토 |
| 7 | **급성 방사통 활동기** | 신경 자극 활동기 — manipulation 자극에 민감 | Manipulation 신중 · Mobilization·운동 우선 · 진행 시 의뢰 |
| 8 | **PFP (슬개대퇴 통증) 단독 도수 적용** | 단독 도수치료 효과 제한적 (CPG 근거) | Hip+Knee 운동 결합 권장 — 단독 도수 비권장 |

## 🩺 보조 스크리닝 도구 (선택)

- **경추 (Cervical)**: 5D's And 3 N's — Dizziness, Drop attacks, Diplopia, Dysarthria, Dysphagia + Ataxia, Nystagmus, Numbness, Nausea
- **요추 (Lumbar)**: Cauda equina 5 항목 — 안장 부위 감각, 양측 하지 증상, 방광 기능, 항문 괄약근, 성기능
- **어깨 (Shoulder)**: Drop arm test (회전근개 전층 파열 의심)
- **무릎 (Knee)**: Ottawa knee rule (외상 후 체중부하 불가)
- **공통**: 야간통 + 체중감소 + 발열 동반 시 비기계적 통증으로 분류

## 🔁 의뢰 흐름 권장 문장 (환자 안내 예시)

> "지금 보이는 신호는 도수치료 전에 의사 선생님 진료를 한 번 받아보시는 게 좋겠어요. 큰 문제가 아닐 수도 있지만, 먼저 확인하고 진행하는 게 안전합니다."

- 환자의 불안을 키우는 단정 표현 (예: "이건 OO 입니다")은 피합니다.
- "확인 후 진행"이라는 절차적 안내로 자연스럽게 의뢰합니다.
- 추천 카드의 의뢰 안내 모달 문구를 그대로 활용해도 됩니다.
